# International Brand Tracker Website

This is a static dashboard for the AU, Canada, and UK international brand tracker.

Open `index.html` in a browser to view the site. All chart and narrative data lives in `data.js`.

For the next biannual update:

1. Add the new wave name to `waves` and update `latestWave`.
2. Add unaided and aided awareness data under `awareness`.
3. Add Shapermint trend metrics under `shapermint`.
4. Update the `latest` market diagnostics and `crossMarketInsights`.

The UI renders from `data.js`, so no HTML changes are needed for routine tracker updates.
